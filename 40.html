Основы работы с коллекциями Map

Задача 1. Пусть даны 3 массива. Создайте коллекцию Map, сделайте ключами коллекции эти массивы, а значениями - какие-нибудь строки.

Задача 2. Переберите циклом for-of коллекцию, созданную в предыдущей задаче.

Задача 3. Пусть дана коллекция Map. С помощью метода keys получите массив ее ключей и переберите его циклом for-of.

Задача 4. Пусть дана коллекция Map. С помощью метода values получите массив ее значений и переберите его циклом for-of.

Задача 5. Пусть дана коллекция Map. С помощью свойства size узнайте количество элементов в этой коллекции.

Задача 6. Пусть даны абзацы. Переберите эти абзацы циклом и создайте коллекцию Map, ключами в которой будут абзацы, а значением - их порядковый номер на странице. Сделайте так, чтобы по клику на любой абзац ему в конец записывался его порядковый номер.

Основы работы с коллекциями Set

Задача 1. Создайте пустую коллекцию Set. С помощью метода add добавьте в нее элементы со значениями 1, 2 и 3.

Задача 2. Создайте коллекцию Set с начальными значениями 1,2,3. С помощью свойства size выведите на экран количество элементов в коллекции.

Задача 3. Создайте коллекцию Set с начальными значениями 1,2,3. С помощью метода has проверьте наличие элемента со значением 3, а затем элемента со значением 4.

Задача 4. Создайте коллекцию Set с начальными значениями 1,2,3. С помощью метода add добавьте в коллекцию еще одно число 2. Выведите содержимое коллекции в консоль, убедитесь, что число 2 не добавилось второй раз.
